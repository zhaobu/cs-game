#mahjong-sensitive
敏感词过滤算法

## 概述
* `此服务只能内网访问`
* 此服务提供：
    * 铭感词替换，将所有敏感词替换成**
    * 敏感词检测，检测是否有敏感词，0表示不含敏感词，1表示含敏感词

## 接口
### 敏感词替换
* 接口地址： http://ip:53782/replace
* 接口参数：接受名为“s”的参数，接受post、get两种方式
* 接口返回值：替换后的字符串


### 敏感词检测
* 接口地址： http://ip:53782/check
* 接口参数：接受名为“s”的参数，接受post、get两种方式
* 接口返回值：
    * "0"：不含敏感词
    * "1"：含敏感词
